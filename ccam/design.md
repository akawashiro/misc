このディレクトリにある ccam.pdf を読んで、ML^□ を CCAM で実行できるようにコンパイルし、また CCAM 上で実行する web アプリケーションを作って。

# 全体の構成
TypeScript + React の web アプリケーションとして作って。
npm create vite@latest でディレクトリを作るところから実装作業を初めて。

動作はクライアントサイドだけで完結させて。
画面は ML^□ を入力するところと、コンパイル過程を表示するところ、CCAM 上での実行を表示するところに分けて。

ML^□ のパース、コンパイル、実行は React なしでもテストできるのでテストを書いて。
たぶん vitest というのがあったと思う。

# コンパイル
コンパイルはその過程が一行づつ追いやすいようにして。
過程は一度にすべて表示して。

# 実行
CCAM のスタックとプログラムを一遷移ずつ表示して。
すべての遷移を同じ画面に表示して。
クリックなどで一遷移するような機能はいらない。
